01Company context

Cloudflare 靠「免費層養網路、網路養安全、安全養平台」長成代理全球五分之一 web 流量的基礎設施商，現在押第四幕：當 AI 爬蟲與 agent 流量在 2027 年超過人類，它要當收費站——預設封鎖 AI 爬蟲、讓出版商對每次抓取開價、自己當清算行。張力在於敘事層跑在損益層前面：pay-per-crawl 還沒入帳一毛錢，核心收入仍是安全與平台訂閱，而 2025/11 的全球大當機正好出在收費站賴以運作的 bot 判讀能力上。以下照 note 1 的 BCM 輸入欄位整理；pace／AI 疊加層留到圖上。

02Business ObjectivesBCM top row

  • 營收衝 $5B annualized run-rate（2028）2026 過 $3B run-rate、FY2026 指引 $2,805–2,813M（~30% 成長）；Rule of 40 >46%、明年看 >50%〔earnings call〕
  • 大客戶平台化（多產品綁定）>$100k 客戶 4,416 家（+25%）、佔營收 72%；DNR 118%；$1M+ 合約數 +73%〔earnings call〕
  • 把開發者平台變 AI 應用預設開發者 5.5M（單季 +1M）；Workers AI 推論請求 +4,000% YoY；某 AI lab 15 天 0→100 萬 Workers 實例〔earnings call〕
  • 建成 agentic web 收費與內容市場（第四幕）pay-per-crawl 尚無入帳收入——目前只有 $3.1M/3 年爬蟲管控合約這類代理指標；管理層估非人流量 2027 超越人類〔earnings call，敘事先行〕
  • AI-first 營運槓桿裁 20%（~1,100 人、費用 $140–150M）；97% 工程師用 AI 工具；非 GAAP 營益率 14%→指引續升〔8-K／earnings call〕

05Data notes

  • 最大的上色陷阱：pay-per-crawl 還不是「會賺錢的能力」——財報沒有這個收入科目，它目前只是 bot 管理＋計費兩條既有能力的重組。toll authority 是敘事層，不是損益層；圖上給 differentiating 是押注，不是現況。
  • 兩個前提修正：FY2025 營收實為 $2,167.9M +29.8%（不是 ~$2.09B）；「$15M Workers AI 合約」查無實據，可考的是「>$100M 史上最大合約、由 Workers 驅動」〔8-K〕。
  • 結構性反諷要保留：2025/11/18 全球當機的禍首（bot 判讀 feature file 管線）正是收費站賴以運作的能力——韌性工程那格不是湊數，是第四幕的前提。
  • 批評面：估值約 28x NTM EV/Revenue 遠高同業；出版商側 CJL「Same Gatekeepers, New Tollbooths」（2026/4）點名「基礎設施守門人自己開市場」的利益衝突；$85M 單一 AI 客戶有集中風險；裁 20% 的「AI 營運槓桿」說法，trade press 仍存疑。
  • 視角提醒：同一個 pay-per-crawl，對 Cloudflare 是 capability、對出版商是分發通路、對 AI 公司是成本牆——這張圖釘死在 Cloudflare 自己的視角。
  • pace／AI 疊加層不放清單，到 BCM 圖才上色；內部 metadata（owner／systems／data in-out／readiness 1–5）非公開、全標待補。
